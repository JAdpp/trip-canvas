# Comic Panel Patterns

Use this when the user asks for 小漫画, manga diary, character interaction, funny scenes, or emotional episode storytelling.

## Panel Types

- `establishing panel`: place or atmosphere, often a landmark or room.
- `reaction panel`: character face, surprise, joy, tiredness, awe.
- `object close-up`: ticket, food, artifact, plush toy, phone photo, stamp.
- `action panel`: walking, riding, eating, pointing, taking photos, running, resting.
- `dialogue panel`: speech bubble, shared joke, short quote.
- `closing mood panel`: night view, back view, souvenir, quiet ending.

## Common Structures

### 3-Panel Tiny Episode

1. Setup: where and what is happening.
2. Reaction: character feeling or comic turn.
3. Memory tag: caption, object, or closing view.

### 4-Panel Daily Strip

1. Morning expectation.
2. Main encounter.
3. Small problem or surprise.
4. Warm/funny resolution.

### Scrapbook With Embedded Panels

Use one large photo/illustration anchor and 2-4 small comic panels around it. Good for preserving real travel evidence while adding character story.

## Dialogue Rules

- Keep speech bubbles short: 4-12 Chinese characters when possible.
- Use exact user phrases if provided.
- Mark invented dialogue as `建议文案`.
- Avoid long exposition inside generated images; put longer writing in a separate text list for overlay.

## Prompt Language

Use phrases such as:

`small manga diary panels`, `cute chibi reaction stickers`, `speech bubbles with short handwritten Chinese`, `mini-comic vignettes integrated into a travel scrapbook page`, `characters interact with tickets/photos/food as memory props`.
