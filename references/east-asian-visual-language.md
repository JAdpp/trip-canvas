# East Asian Visual Language

Use this before page planning or prompt writing.

## Core Positioning

The target is East Asian sticker-style travel memory art: dense, cute, layered, handwritten, object-rich, and emotionally captioned. It should feel closer to 手帐, 贴纸拼贴, 咕卡, 小红书旅行视觉笔记, and light manga diary than to a clean Western photobook or route board.

## Visual Grammar

- Layered paper collage: washi tape, torn paper, grid paper, cream notebook paper, ticket fragments, stamp marks.
- Sticker cutouts: white border photos, food stickers, museum artifact stickers, transport icons, tiny hearts, stars, check marks.
- Handwritten Chinese: short captions, date badges, speech bubbles, checklist notes, tiny mood comments.
- Cute density: many small memory fragments, but keep clusters legible.
- Character presence: recurring travelers may be central actors, not only corner mascots.
- Small comic rhythm: reaction faces, speech bubbles, motion lines, "today's tiny episode" panels.

## Information Hierarchy

1. Page title/date and trip theme.
2. Emotional/memory anchor scene.
3. Main photos or generated illustrations.
4. Character/comic moments.
5. Captions, speech bubbles, handwritten notes.
6. Tickets, maps, receipts, route snippets, stamps.
7. Decorative stickers.

Travel facts support the memory; they should not dominate unless the user asks for a practical itinerary page.

## Layout Patterns

### Three-Day Wide Spread

Use vertical day columns. Each day has one large anchor visual, 2-4 supporting clusters, a caption block, and one mini-comic or character moment.

### Phone Long Page

Use top-to-bottom memory flow with large Day badge, alternating image clusters and caption cards. Keep text large enough to read.

### Notebook Spread

Use left/right pages, tape, stamps, torn paper, and physical scrapbook cues. Good for printable or desktop viewing.

### Mini-Comic Hybrid

Use 3-8 panels for character action, plus photo/ticket stickers around panels.

## Negative Rules

- Do not make it a clean travel itinerary board unless requested.
- Do not reduce the page to a generic mood collage with no memory objects.
- Do not make every element symmetrical or grid-perfect.
- Do not overload tiny Chinese paragraphs that image models will distort.
- Do not use a single dominant palette if the trip has varied scenes; allow theme colors per day or place.
- Do not make characters so large that photos, tickets, or place memories disappear.
