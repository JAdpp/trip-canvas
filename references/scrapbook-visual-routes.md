# Scrapbook Visual Routes

Use this before page planning or prompt writing.

## Core Positioning

The target is memory-first travel scrapbook and visual journal art: layered, object-rich, emotionally captioned, and visibly made from trip materials. It may be East Asian sticker-style, watercolor, vintage, urban sketch, polaroid-led, map-led, minimalist, comic-led, or hybrid. It should feel closer to handmade travel memory-making than to a clean itinerary board, generic poster, or ordinary photo book.

## Shared Visual Grammar

- Layered memory objects: photos, ticket fragments, receipts, maps, stamps, stickers, handwritten notes, and paper textures.
- Replaceable slots: keep source photos, text cards, character stickers, map panels, and decorative objects visually separable.
- Clear memory hierarchy: emotion and scene first, travel logistics second unless the user asks for a practical itinerary.
- Short readable labels: use large captions, date badges, speech bubbles, and mood notes; keep long writing outside the image prompt when fidelity matters.
- Style-specific density: sticker/comic routes can be dense; minimalist and watercolor routes need more air.
- Personal continuity: recurring people, pets, or avatars should stay consistent when used, but photo-led routes may keep characters minimal.

## Visual Routes

Choose a visual route before prompt writing. The route controls rendering language; the editable object manifest stays the same.

| Route | Use When | Visual Notes |
| --- | --- | --- |
| `East Asian sticker comic` | the user wants character-heavy, playful, dense scrapbook pages with handwritten labels | white-border sticker cutouts, washi tape, grid paper, light manga or chibi diary cues, food/souvenir stickers, speech bubbles |
| `kawaii sticker journal` | the memory should feel bright, soft, cute, or youthful without needing a specifically East Asian layout | rounded icons, soft characters, pastel sticker sheets, cheerful labels, playful framing |
| `black-and-white cartoon doodle` | the memory should feel light, printable, quick, or hand-drawn | pure black marker lines, simple rounded cartoon figures, icons, blank caption cards, generous white space; avoid grayscale detailed manga or realistic shading |
| `map infographic scrapbook` | route sequence, geography, or place relationships matter | illustrated route map as a main object, numbered stops, transport icons, photo slots, concise labels |
| `polaroid photo collage` | real photos should dominate and characters should recede | instant-photo frames, taped snapshots, film grain, pressed flowers, map scraps, handwritten captions |
| `watercolor travel journal` | scenery, gardens, coasts, historic streets, or slow travel mood should dominate | translucent color washes, ink outlines, soft paper, airy margins, gentle caption cards |
| `vintage ephemera scrapbook` | the user wants nostalgia, old streets, rail travel, museums, or analog diary feeling | aged paper, postcards, receipts, stamps, muted palette, serif labels, worn tape, archival scraps |
| `urban sketch journal` | city walks, architecture, cafes, street scenes, or observational travel notes matter | loose pen-and-wash sketches, location annotations, street textures, imperfect linework |
| `minimalist line journal` | the user wants a clean, printable, modern, or calm travel memory page | simple line art, sparse color accents, generous white space, restrained typography |
| `comic book travel page` | the trip has action, humor, dialogue, or character-heavy moments | panel gutters, action lines, speech bubbles, reaction faces, clear episode sequence |
| `mixed-media collage` | the user wants expressive art-journal energy rather than a tidy page | torn paper, magazine-like cutouts, tape, ink marks, varied textures, layered but still readable |
| `hybrid scrapbook-comic` | the page needs both factual structure and emotional mini-scenes | combine a main map/photo/sketch route with 2-4 small comic panels or sticker clusters |

Avoid making every public-safe example or user page look like the same colorful chibi manga collage. Keep the memory structure and object IDs consistent while varying the rendering route.

## Information Hierarchy

1. Page title/date and trip theme.
2. Emotional/memory anchor scene.
3. Main photos or generated illustrations.
4. Character/comic moments when the selected route uses characters.
5. Captions, speech bubbles, handwritten notes.
6. Tickets, maps, receipts, route snippets, stamps.
7. Decorative stickers or texture.

Travel facts support the memory; they should not dominate unless the user asks for a practical itinerary page.

## Layout Patterns

### Multi-Day Wide Spread

Use vertical day columns. Each day has one large anchor visual, 2-4 supporting clusters, a caption block, and optionally one mini-comic or character moment.

### Phone Long Page

Use top-to-bottom memory flow with a large day/date badge, alternating image clusters and caption cards. Keep text large enough to read.

### Notebook Spread

Use left/right pages, tape, stamps, torn paper, and physical scrapbook cues. Good for printable or desktop viewing.

### Mini-Comic Hybrid

Use 3-8 panels for character action, plus photo/ticket/map stickers around panels.

### Photo-Led Social Page

Use a small set of dominant photo frames with sparse stickers and captions. Good when the user provides strong photos and wants a social-ready image.

## Negative Rules

- Do not make it a clean travel itinerary board unless requested.
- Do not reduce the page to a generic mood collage with no memory objects.
- Do not make every element symmetrical or grid-perfect unless the selected route is explicitly minimalist.
- Do not overload tiny paragraphs that image models will distort.
- Do not force chibi characters into map-led, photo-led, minimalist, watercolor, or doodle routes unless the user asks.
- Do not treat the black-and-white route as a color image converted to grayscale; it should be simple cartoon doodle line art.
- Do not use a single dominant palette if the trip has varied scenes; allow theme colors per day or place.
- Do not make characters so large that photos, tickets, or place memories disappear.
